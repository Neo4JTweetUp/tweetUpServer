Tweet-Up 
========

Intro
-----
This is a public repo built by Hack Reactor students as a short-project.

Description
-----------
Tweet-up is a mobile app that makes friend recommendations to users, based on their current locations and common friends on Twitter.

Login with your Twitter account to get started.  Based on the common friends and locations, the app will give user recommendations in your network, that might be similar to you in interests. You can immediately send messages to the individuals to connect.

Tech stack
----------
Tweet-up uses Ionic/Angular, Node.js and Neo4J Graph Database.
